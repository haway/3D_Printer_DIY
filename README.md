## 準備工作 ##
* 收集五金零件
* 收集模具零件
* 收集機械零件
* 收集電子零件

### 收集五金零件 ###

五金零件在露天上面都可以買的到, 也有人專門幫你都收集好了, 
如果想要自己購買的話可以參考文件的零件表, 照表的編號跟數量直接買下去就對了
譬如: M3x10mm 6 個, 就直接在露天打 M3x10mm 就會找到零件
零件表中的所有零件都要買到.
